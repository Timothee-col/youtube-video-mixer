# YouTube Video Mixer Pro 🎬

Créez des vidéos TikTok/Reels professionnelles avec reconnaissance faciale IA !

## 🚀 Installation (Local)

1. **Clone the repository:**
```bash
git clone https://github.com/Timothee-col/youtube-video-mixer.git
cd youtube-video-mixer
```

2. **Create a virtual environment:**
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

3. **Install dependencies:**
```bash
pip install -r requirements.txt
```

4. **Run the app:**
```bash
streamlit run youtube_mixer.py
```

## ✨ Fonctionnalités

- 🎯 **Reconnaissance faciale** - Détectez et priorisez une personne spécifique
- 📹 **Format vertical 9:16** - Optimisé pour TikTok/Instagram Reels
- 🎬 **Montage intelligent** - Extraction automatique des meilleurs segments
- 🖼️ **Logo personnalisé** - Ajoutez votre marque avec contrôles précis
- 🎵 **Audio personnalisé** - Remplacez la bande sonore
- 📱 **Tagline vidéo** - Ajoutez une conclusion
- 🔀 **Mélange intelligent** - Alterne les sources automatiquement

## 🎛️ Utilisation

1. **[Optionnel]** Uploadez une photo pour la reconnaissance faciale
2. **Configurez** la durée et les paramètres selon vos besoins  
3. **Collez** vos URLs YouTube (une par ligne)
4. **Personnalisez** avec logo/audio/tagline
5. **Cliquez** sur "Créer la vidéo" et patientez
6. **Téléchargez** votre vidéo verticale optimisée !
